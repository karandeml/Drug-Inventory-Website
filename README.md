# Drug Inventory and Supply Chain Management System

## 📌 Project Overview

The **Drug Inventory and Supply Chain Management System** is a web-based application developed using **PHP, MySQL, HTML, CSS, and Bootstrap**. The system helps manage pharmaceutical products, track stock availability, and handle product ordering efficiently. It is designed to simplify inventory monitoring and improve supply chain operations.

## 🎯 Objective

* To manage drug inventory efficiently.
* To track available stock and product details.
* To maintain accurate records of medicines and orders.
* To provide an easy-to-use interface for users and administrators.

## ⚙️ Features

* Product listing with brand, price, and stock availability
* Inventory management system
* Product image display
* Order management
* Dynamic homepage with product carousel
* Contact and About pages
* Responsive user interface using Bootstrap

## 🛠️ Technologies Used

* **Frontend:** HTML, CSS, Bootstrap, JavaScript
* **Backend:** PHP
* **Database:** MySQL
* **Server:** XAMPP / Apache

## 📂 Project Structure

* `index.php` – Main entry point of the system
* `home.php` – Displays product listings and featured products
* `about.php` – Shows information about the system
* `contact.php` – Contact details and communication information

## 💡 System Benefits

* Helps manage medicine stock effectively
* Reduces manual inventory errors
* Improves supply chain monitoring
* Provides quick access to product information

## 🚀 Future Enhancements

* User authentication system
* Online ordering and payment integration
* Admin dashboard analytics
* Real-time stock alerts

## 📄 Conclusion

This system provides a simple and efficient solution for managing drug inventory and supply chain operations. It improves the accuracy of stock tracking and enhances the overall management of pharmaceutical products.
